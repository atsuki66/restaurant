# 🍷 La Maison — Сайт ресторана с бронированием столиков

Университетский проект: веб-приложение для бронирования столиков в ресторане.

## 🛠 Технологии

- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **Backend:** Python, Flask, Flask-CORS
- **База данных:** JSON файл (`database.json`)

## 📋 Возможности

- 🪑 Выбор из 6 столиков (от 2 до 6 мест)
- 📝 Форма бронирования с валидацией (имя, телефон, дата, время, комментарий)
- ⏰ Проверка что дата/время бронирования — в будущем
- 🔒 Админ-панель для просмотра и отмены бронирований
- 📊 Статистика бронирований
- 🔍 Фильтрация по столикам
- 💾 Хранение данных в JSON файле

## 🚀 Запуск

### 1. Установка зависимостей Python

```bash
pip install -r requirements.txt
```

### 2. Сборка фронтенда

```bash
npm install
npm run build
```

### 3. Запуск сервера

```bash
python server.py
```

### 4. Открыть в браузере

Перейти по адресу: **http://localhost:5000**

## 📡 API Endpoints

| Метод  | URL                            | Описание                    |
|--------|--------------------------------|-----------------------------|
| GET    | `/api/reservations`            | Получить все бронирования   |
| POST   | `/api/reservations`            | Создать бронирование        |
| GET    | `/api/reservations/<id>`       | Получить бронирование по ID |
| DELETE | `/api/reservations/<id>`       | Удалить бронирование        |

### Пример тела POST-запроса:

```json
{
  "tableId": 1,
  "name": "Иван Иванов",
  "phone": "+7 (999) 123-45-67",
  "date": "2025-12-25",
  "time": "19:00",
  "comment": "Столик у окна, пожалуйста"
}
```
